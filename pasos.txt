Trabajar con git:

1. git init
Para inicializar repositorio git, solo se hace una vez en un sitio web

git remote add origin "link del repositorio"

2. git status
Es para chequear que todo este bien

3. git add
Esto permite agregar las modificaciones que agamos hecho

4.git commit -m "Un mensaje cualquiera" (primer commit:git commit -m "first commit")
Esto es para confirmar la subida de repositorio 

git push -u origin main

5. git branch 
Es para crear una rama

6. git checkout "rama a eleccion"
Es para cambiar de rama en rama

7.git log
Es para ver los cambios realizados 

8.git log --oneline 
para ser mar especifico en ver todos los cambios

9. git checkout "codigo de un commit"
Es para ver un commit en especifico, moverme en cada cambio que se hizo
